GitHub API Integration

Sichere Token-Handhabung

Diese Web-App erlaubt das Einfügen eines Iframe-Spielercodes direkt auf der Website. Mit einem Klick wird der Code automatisch ins GitHub-Repo übernommen, damit Vercel es neu deployed.

⚠️ Wichtiger Sicherheitshinweis

Das GitHub Token ist niemals im Client sichtbar! Die API-Authentifizierung muss serverseitig erfolgen:

// Serverseitige API-Route (z.B. /api/github-update) const { Octokit } = require('@octokit/rest'); const octokit = new Octokit({ auth: process.env.GITHUB_TOKEN // Aus Umgebungsvariablen! }); // Repository-Update await octokit.rest.repos.createOrUpdateFileContents({ owner: 'USER', repo: 'REPO', path: 'index.html', message: 'Automatisches Update via Player-Web-App', content: Buffer.from(htmlContent).toString('base64') });

🔧 Vercel Umgebungsvariablen

Fügen Sie in Ihren Vercel-Projekteinstellungen hinzu:

GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxx GITHUB_OWNER=IhrUsername GITHUB_REPO=IhrRepoName
